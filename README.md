# DA280A-Webbutveckling-1
Distanskurs på Malmö universitet: http://mah-webbutv.github.io/courses/da280a/

I denna kurs introducerades webbutveckling genom HTML5 och CSS3. Kursens första del fokuserade på att skapa grundläggande struktuer på webbsidor genom bl.a. listor, tabeller och formulär (inl 1, 2 & 3). Den senare delen bytte fokus mot skapande av layout samt formgivning genom CSS3. Kursen avslutades med en introduktion till responsiv design för att anpassa sin webbplats till olika enheter (inl 4, 5 & 6).

---

**Inlämningsuppgift 1**

I första uppgiften så fick man bekanta sig lite med HTML och komma igång med språket. 

**Inlämningsuppgift 2**

I andra uppgiften fick man lära sig mer om tabeller, bilder och länkar i HTML.

**Inlämningsuppgift 3**

I tredje uppgiften fick man lära sig mer om formulär och hur de fungerar i HTML.

**Inlämningsuppgift 4**

I fjärde uppgiften så fick man bekanta sig lite med CSS och CSS-egenskaper. 

**Inlämningsuppgift 5**

I femte uppgiften fick man lära sig hur man skulle strukturera upp en sida med CSS.

**Inlämningsuppgift 6**

I sjätte uppgiften fick man lära sig ännu mer om struktura upp en sida men även hur man ska använda responsiv design i CSS.
